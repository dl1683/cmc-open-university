What is this?

A free, open educational resource. Pick any topic, press play, and watch the algorithm run — with a synchronized explanation of what is happening and why at every single step. Change the inputs and run it again. Read the source: every algorithm is written to be read, at github.com/dl1683/DataStructuresInJavaScript.

New topics are added over time — data structures, classic algorithms, core concepts, and modern AI.

The site is arranged into guided curriculum tracks so learners can study in connected progressions. Tracks can now depend on prior tracks, keep checkpoint prompts, and surface the next path once a path is complete.

Use the topic page map to see exactly which concepts feed into this one, what each path unlocks, and what to study next in the same learning thread.
